Detailed Description

template<class Value = bool>
class irk::mutable_bit_trie< Value >

In-memory, mutable implementation of Bitwise Trie.

This implementation is not optimized for efficiency (doesn't necessarily mean it isn't for most applications). Instead, it is meant to be used mainly for building tries, or whenever a mutable version is needed.

In particular, this implementation is not conservative when it comes to memory usage. Ease of use is a priority here.
