Constructs a zip view.

Template Parameters
ZipFnA function type for element generation.
LeftRangeThe type of the left range.
RightRangeThe type of the right range.
Parameters
leftThe left range.
rightThe right range.
zip_fnA function which, given an element of left and an element of right, returns an element of the zipped view.

Example:

std::vector<int> documents = {0, 1, 3};
std::vector<double> scores = {1.2, 3.1, 0.4};
auto zip_view = zip(documents, scores,
[](const int& doc, const double& score) {
return std::make_pair(doc, score);
});
for (const auto& [doc, score] : zip_view) {
std::cout << doc << ", " << score << std::endl;
}
