Detailed Description

template<class Symbol = char, class Ptr = std::uint16_t, class MemoryContainer = std::vector<char>>
class irkit::io::CompactABST< Symbol, Ptr, MemoryContainer >

Read-only array-based representation of an Alphabetic Binary Search Tree.

Template Parameters
Symbolsymbol type, char by default
Ptrthe type of the children pointers; determines the maximum size of the tree;
MemoryContainermemory container type, std::vector<char> by default; must implement data() and size(); data() should implement operator[]
