◆ next_doc()

template<class Range >
posting_type irk::union_range< Range >::next_doc ( )
inline

Returns the next accumulated posting in sorted union order.

An accumulated posting is an object of the same type as posting, but its score is a sum of all the partial scores for the next available document.

Note that if next_posting is invoked independently, this will either return an incorrect score or will skip a document (or documents).
