Detailed Description

template<class Doc = long, class Term = std::string, class TermId = long, class Freq = long>
class irk::index::index_assembler< Doc, Term, TermId, Freq >

Builds an index in batches and merges them together on disk.

See assemble() function documentation for the format of the input file. Note that neither the assembler nor the builder stem terms. It must be done beforehand.
