Detailed Description

template<class DocumentCodec = irk::stream_vbyte_codec<index::document_t>, class FrequencyCodec = irk::stream_vbyte_codec<index::frequency_t>>
class irk::index::basic_index_assembler< DocumentCodec, FrequencyCodec >

Builds an index in batches and merges them together on disk.

See assemble() function documentation for the format of the input file. Note that neither the assembler nor the builder stem terms. It must be done beforehand.
