Detailed Description

template<class T, class Codec = varbyte_codec<T>, class MemoryBuffer = std::vector<char>>
class irk::compact_table< T, Codec, MemoryBuffer >

Fast-access compressed array.

Template Parameters
Tthe type of the stored values
MemoryBufferthe memory buffer type, e.g., std::vector<char> (for building/reading) or boost::iostreams::mapped_file (for reading)
Codeca codec that encodes values of type T

This is a compressed table indexed with consecutive integers between 0 and size - 1. TODO: describe implmentation.
