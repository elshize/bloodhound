Detailed Description

template<class Symbol = char, class MemoryContainer = std::vector<char>>
class irk::hutucker_codec< Symbol, MemoryContainer >

Hu-Tucker codec.

Template Parameters
Symbolthe type of encoded symbols (char by default)
MemoryContainerwhere the ABST memory is read from; could be, e.g., a vector or a memory mapped file

See http://www-math.mit.edu/~shor/PAM/hu-tucker_algorithm.html for a description of the algorithm and sturctures.
